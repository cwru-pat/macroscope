How to contribute
*****************

We welcome contributions from anyone via pull requests on `GitHub
<https://github.com/cwru-pat/macroscope>`_. If you don't feel comfortable modifying or
adding functionality, we also welcome feature requests and bug reports as
`GitHub issues <https://github.com/cwru-pat/macroscope/issues>`_.
